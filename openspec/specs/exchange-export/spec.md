# exchange-export

Выгрузка данных на базе-источнике по правилам обмена.

## Requirements

### Requirement: Выгрузка через обработку ргУниверсальныйОбменXML

На базе-источнике выгрузка SHALL выполняться обработкой `ргУниверсальныйОбменXML` расширения — единым движком на базе штатной `УниверсальныйОбменДаннымиXML` ЗУП с доработками по ТЗ.

#### Scenario: Загрузка правил из макета

- **WHEN** сервер получает имя макета правил в запросе GetData
- **THEN** система загружает правила обмена из указанного макета конфигурации-источника

### Requirement: Инициализация параметров правил

Система SHALL инициализировать параметры правил обмена значениями, полученными от веб-сервиса (из XDTO-структуры).

#### Scenario: Подстановка параметров перед выгрузкой

- **WHEN** выгрузка запускается с набором параметров из GetData
- **THEN** параметры обработки обмена заполняются этими значениями до выполнения выгрузки

### Requirement: Активация всех ПВД

Система SHALL активировать все правила выгрузки данных (ПВД), содержащиеся в загруженных правилах обмена.

#### Scenario: Полная выгрузка по правилам

- **WHEN** выполняется выгрузка по загруженным правилам
- **THEN** все ПВД из правил участвуют в формировании XML-файла

### Requirement: Без автоматического поиска ссылок

Система MUST NOT выполнять автоматический поиск ссылочных объектов по примитивным параметрам. Преобразование примитивов в ссылки SHALL выполняться только в обработчиках событий правил обмена.

#### Scenario: Примитивный параметр без обработчика в правилах

- **WHEN** параметр типа строка передаётся в правило, а обработчик преобразования в ссылку не реализован в правилах
- **THEN** ядро не подставляет ссылку автоматически; поведение определяется правилами обмена или фиксируется ошибка движка

### Requirement: Упаковка результата в ZIP

Результат выгрузки (XML-файл) SHALL автоматически упаковываться в ZIP-архив перед возвратом клиенту.

#### Scenario: Сжатие выгрузки

- **WHEN** выгрузка успешно сформировала XML-файл
- **THEN** файл упакован в ZIP для передачи через GetData
