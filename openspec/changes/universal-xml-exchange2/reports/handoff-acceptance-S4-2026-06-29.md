## Срез S4 — передача на приёмку: universal-xml-exchange2

**Прогресс:** 8/8 рабочих задач [x]; `S4.accept` — `[ ]` (ручная приёмка).

### 1. Что реализовано

Перечисление статусов удалено из метаданных расширения; реквизит `Статус` — строка с кодами из серверного модуля. На форме профиля-источника статус только для просмотра, управление — кнопками «Подготовить к обмену» и «Отменить обмен»; при отмене из «Идёт обмен» — диалог подтверждения. В объекте — guard от ручной смены статуса на серверные коды.

### 2. Что проверить СЕЙЧАС

1. Профиль-источник с правилами и префиксами → **«Подготовить к обмену»** → статус «Готов к обмену» → обмен с приёмника успешен (регрессия S2).
2. **«Отменить обмен»** → статус «Не готов»; повторный GetData без подготовки отклоняется.
3. Смоделировать «Идёт обмен» (прервать сеанс или оставить зависший) → **«Отменить обмен»** → диалог → после «Да» сброс в «Не готов».
4. Прямое редактирование поля статуса недоступно; попытка обойти через запись объекта с кодом «Выполняется» — отказ guard.

Опционально: удалить из репозитория orphan `Enums/рг_СтатусыОбмена.xml` (в конфигурации enum уже нет).
