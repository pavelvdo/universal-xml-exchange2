# Приложение 3. Правила формирования текстов стандартных команд и автоматических заголовков форм

> **Agent ref:** Правила формирования текстов стандартных команд и автоматических заголовков форм в 1С:Предприятие. Свойства метаданных для представлений и заголовков.
> **Ключевые разделы:** команды, заголовки форм, представление списка/объекта, расширенное представление, пояснение.

---

## Свойства метаданных для представления

В свойствах метаданных задаются: **Представление объекта** (для регистра — записи), **Расширенное представление объекта** (записи), **Представление списка**, **Расширенное представление списка**, **Пояснение**, **Картинка**.

Объекты конфигурации, у которых поддерживаются эти свойства (кратко): подсистемы, константы, планы обмена, критерии отбора, общие формы, справочники, документы, журналы документов, перечисления, отчёты, обработки, планы видов характеристик, планы счетов, планы видов расчёта, регистры сведений (подчинённые и независимые), регистры накоплений, бухгалтерии, расчёта, бизнес-процессы, задачи. Для части объектов доступны не все свойства (например, только представление списка и расширенное представление списка).

---

## Правила формирования представления и подсказки команд

Обозначение: **Свойство1 → Свойство2** — если Свойство1 не задано, используется Свойство2.

| Действие | Представление команды | Подсказка команды |
|----------|------------------------|-------------------|
| Открытие списка | Представление списка | Пояснение → Расширенное представление списка → Представление списка |
| Создание нового объекта (записи) | Представление объекта: создать / Представление записи: создать | Расширенное представление объекта (записи) → Представление объекта (записи) |
| Создание новой группы | Представление списка: создать группу | Расширенное представление списка → Представление списка; при отсутствии — Пояснение → Расширенное представление списка → Представление списка |
| Ввод на основании | Представление объекта: создать на основании | Расширенное представление объекта → Представление объекта |
| Открытие отчёта или обработки | Представление отчёта/обработки | Пояснение → Расширенное представление |
| Открытие общей формы | Представление общей формы | Пояснение → Расширенное представление |
| Открытие формы редактирования константы | Представление константы | Пояснение → Расширенное представление |
| Переход к подсистеме (панель разделов) | Представление подсистемы | Пояснение; графика: Картинка → стандартная картинка раздела |

**Примечание.** Если все используемые свойства не заданы — используется текстовое представление объекта метаданных (метод `Представление()`).

### Панель действий и командная панель формы

Из текста представления команды удаляется название группы команд (предваряемое символом «:»), если это сочетание является окончанием представления. Пример: «Товары: создать» в панели действий отображается как «Товары». Если команда «Наша команда: создать на основании» помещена в стандартную группу «Создать на основании» — отображается «Наша команда»; в пользовательской группе «Печать» — «Наша команда: создать на основании» сохраняется.

## Автоматический заголовок формы

Если **Автоматический заголовок формы** = Истина, заголовок = объединение свойства **Заголовок формы** и автоматически формируемой части. Если Ложь — используется только свойство Заголовок формы без преобразований.

| Контекст формы | Существующий элемент | Новый элемент |
|----------------|---------------------|---------------|
| Форма списка | — | Расширенное представление списка → Представление списка |
| Справочник, план счетов, план видов расчёта, план обмена, план видов характеристик, задача | Текстовое представление объекта (Расшир. представление объекта → Представление объекта), напр. «Зиновьев Антон (Сотрудник)», «70.1 (Счет бухгалтерского учета)» | Расширенное представление объекта (Создание) → Представление объекта (Создание) |
| Документ, бизнес-процесс | Текстовое представление объекта, напр. «приходная накладная № 12001 от 12.05.2008 16:15:32» | Расширенное представление объекта (Создание) → Представление объекта (Создание) |
| Запись регистра сведений | Расширенное представление записи → Представление записи | Расширенное представление записи (Создание) → Представление записи (Создание) |
| Группа справочника / плана видов характеристик | Текстовое представление объекта (Расшир. представление списка → Представление списка) | Расширенное представление списка (Создание группы) → Представление списка (Создание группы) |
| Отчёт, обработка | — | Расширенное представление |

Если все используемые в формировании наименования свойства не заданы — используется текстовое представление объекта метаданных.
