# Приложение 6. Описание прав доступа

> **Agent ref:** Описание прав доступа при редактировании ролей в 1С:Предприятие.  
> **Ключевые разделы:** права по объектам, административные права, мобильная платформа.  
> Символом «*» помечены права, поддерживаемые в мобильной платформе.

---

## Общие положения

Права доступа задаются при редактировании ролей. Ниже перечислены права с краткими описаниями.

## Права доступа

**Automation (Automation)** — использование automation-сервера.

**Администрирование (Administration)** — административные действия над всей информационной базой и редактирование списка пользователей (для конфигурации); требует права АдминистрированиеДанных. Включает: настройку параметров соединения внешнего источника данных; отображение списка лицензий в окне «О программе»; удаление областей данных; фоновое обновление конфигурации БД на клиенте; установку имени профиля безопасности в диалогах расширений; расширенный просмотр журнала регистрации; управление копиями БД и обновление копии.

**АдминистрированиеДанных (DataAdministration)** — административные действия над данными: просмотр событий журнала (Аутентификация, Доступ, Провайдер OpenID и др.); снятие ограничений по отборам в журнале (для просмотра журнала нужно право ЖурналРегистрации); установка часового пояса ИБ; создание начального образа подчинённого узла РИБ; обновление нумерации объектов; выполнение методов СтандартноеХранилищеНастроекМенеджер для других пользователей; ПолучитьДанныеРегистрацииИнформационнойБазы/УстановитьДанныеРегистрацииИнформационнойБазы; управление настройками обработки ошибок для других пользователей; создание обсуждения системы взаимодействия в режиме «только просмотр».

**АдминистрированиеРасширенийКонфигурации** — доступ к интерфейсу управления расширениями в режиме «1С:Предприятие». В безопасном режиме недоступно.

**АктивныеПользователи (ActiveUsers)** — просмотр списка активных пользователей.

**ВводПоСтроке* (InputByString)** — использование режима ввода по строке.

**ВебКлиент (WebClient)** — запуск веб-клиента.

**ВнешнееСоединение (ExternalConnection)** — открытие внешнего соединения.

**Вывод* (Output)** — вывод на печать, запись и копирование в буфер обмена.

**Выполнение (Execute)** — выполнение задачи.

**Добавление (Insert)** — добавление объекта (основное).

**ЖурналРегистрации (EventLog)** — просмотр журнала регистрации. Полнота отображения задаётся правами Администрирование и АдминистрированиеДанных.

**ЗавершениеМонопольногоРежимаПриНачалеСеанса (ExclusiveModeTerminationAtSessionStart)** — возможность завершить сеанс с монопольным режимом, если при установке монопольного режима разрешено завершение сеанса.

**Изменение (Update)** — сохранение изменённого объекта (основное).

**ИзменениеАутентификацииОССеанса (SessionOSAuthenticationChange)** — изменение параметров аутентификации ОС внешнего источника данных в настройках пользователя и параметрах сеанса.

**ИзменениеИсторииДанных (UpdateDataHistory)** — изменение истории данных (создание новой версии, удаление версии).

**ИзменениеИсторииДанныхОтсутствующихДанных (UpdateDataHistoryOfMissingData)** — то же для отсутствующих данных (удалённые или недоступные по RLS).

**ИзменениеКомментарияВерсииИсторииДанных (UpdateDataHistoryVersionComment)** — изменение комментария к версии данных.

**ИзменениеНастроекИсторииДанных (UpdateDataHistorySettings)** — изменение настроек истории данных.

**ИзменениеСтандартнойАутентификации (StandardAuthenticationChange)** — изменение сохранённых параметров стандартной аутентификации внешнего источника данных (имя, пароль).

**ИзменениеСтандартнойАутентификацииСеанса (SessionStandardAuthenticationChange)** — изменение параметров стандартной аутентификации внешнего источника данных для текущего сеанса.

**ИнтерактивнаяАктивация (InteractiveActivate)** — интерактивная активация.

**ИнтерактивнаяОтменаПроведения* (InteractiveUndoPosting)** — интерактивная отмена проведения.

**ИнтерактивнаяПометкаУдаления* (InteractiveSetDeletionMark)** — интерактивная пометка на удаление.

**ИнтерактивнаяПометкаУдаленияПредопределенныхДанных* (InteractiveSetDeletionMarkPredefinedData)** — интерактивная пометка удаления предопределённых данных.

**ИнтерактивноеВыполнение (InteractiveExecute)** — интерактивное выполнение задачи.

**ИнтерактивноеДобавление* (InteractiveInsert)** — интерактивное добавление.

**ИнтерактивноеИзменениеПроведенных* (InteractiveChangePosted)** — интерактивное редактирование проведённого документа (удаление, пометка удаления, перепроведение). Без права форма открывается в режиме просмотра.

**ИнтерактивноеОткрытиеВнешнихОбработок (InteractiveOpenExtDataProcessors)** — интерактивное открытие внешних обработок.

**ИнтерактивноеОткрытиеВнешнихОтчетов (InteractiveOpenExtDataReports)** — интерактивное открытие внешних отчётов.

**ИнтерактивноеПроведение* (InteractivePosting)** — интерактивное проведение.

**ИнтерактивноеПроведениеНеОперативное (InteractivePostingRegular)** — интерактивное проведение документа в неоперативном режиме (стандартные команды форм).

**ИнтерактивноеСнятиеПометкиУдаления* (InteractiveClearDeletionMark)** — интерактивное снятие пометки на удаление.

**ИнтерактивноеСнятиеПометкиУдаленияПредопределенныхДанных* (InteractiveClearDeletionMarkPredefinedData)** — интерактивное снятие пометки удаления с предопределённых данных.

**ИнтерактивноеУдаление (InteractiveDelete)** — интерактивное непосредственное удаление.

**ИнтерактивноеУдалениеПомеченных (InteractiveDeleteMarked)** — интерактивное удаление помеченных объектов.

**ИнтерактивноеУдалениеПомеченныхПредопределенныхДанных* (InteractiveDeleteMarkedPredefinedData)** — интерактивное удаление помеченных предопределённых данных.

**ИнтерактивноеУдалениеПредопределенныхДанных (InteractiveDeletePredefinedData)** — интерактивное удаление предопределённых данных.

**ИнтерактивныйСтарт (InteractiveStart)** — интерактивный старт бизнес-процесса.

**Использование (Use)** — использование объекта конфигурации; для функции внешнего источника данных — использование из встроенного языка и языка запросов.

**КлиентСистемыАналитики (AnalyticsSystemClient)** — запуск клиентского приложения «1С:Аналитика» из интерфейса «1С:Предприятие».

**МонопольныйРежим (ExclusiveMode)** — использование монопольного режима.

**ОбновлениеКонфигурацииБазыДанных (UpdateDataBaseConfiguration)** — выполнение обновления конфигурации БД (на сервере или фоновое). Достаточно этого права без Администрирование.

**ОтменаПроведения (UndoPosting)** — отмена проведения документов.

**ПереходНаВерсиюИсторииДанных (SwitchToDataHistoryVersion)** — интерактивное восстановление состояния версионируемых реквизитов на выбранную версию. Без права недоступна команда «Перейти на версию» в форме истории изменений.

**Получение (Get)** — получение значения, не хранящегося в БД.

**Проведение (Posting)** — проведение документов.

**Просмотр* (View)** — просмотр. В диалоге выбора типа недоступны типы без этого права. Для функции внешнего источника — использование в списке полей СКД.

**ПросмотрИсторииДанных (ViewDataHistory)** — интерактивный просмотр истории данных. Без права недоступна команда «История изменений».

**РегистрацияИнформационнойБазыСистемыВзаимодействия (CollaborationSystemInfoBaseRegistration)** — регистрация ИБ в сервисе 1С:Диалог.

**Редактирование* (Edit)** — редактирование.

**РедактированиеКомментарияВерсииИсторииДанных (EditDataHistoryVersionComment)** — интерактивное изменение комментария версии в списке версий истории данных.

**РежимОсновногоОкнаВстроенноеРабочееМесто (MainWindowModeEmbeddedWorkplace)** — запуск основного окна в режиме встроенного рабочего места.

**РежимОсновногоОкнаКиоск (MainWindowModeKiosk)** — запуск в режиме киоска.

**РежимОсновногоОкнаОбычный (MainWindowModeNormal)** — запуск в обычном режиме.

**РежимОсновногоОкнаПолноэкранноеРабочееМесто (MainWindowModeFullscreenWorkplace)** — запуск в режиме полноэкранного рабочего места.

**РежимОсновногоОкнаРабочееМесто (MainWindowModeWorkplace)** — запуск в режиме рабочего места.

**РежимТехническогоСпециалиста* (TechnicalSpecialistMode)** — доступность в меню «Сервис и настройки» команды «Функции для технического специалиста».

**СохранениеДанныхПользователя (SaveUserData)** — возможность сохранения настроек пользователя. При выключении: недоступны история, избранное, настройка форм, настройки панелей и начальной страницы; настройки отчётов и вариантов не сохраняются; параметры и восстановление данных форм недоступны; автосохранение форм отключено; размеры окон и панелей только на сеанс; период в списках не сохраняется; при программном сохранении — исключение.

**Старт (Start)** — старт бизнес-процесса.

**ТолстыйКлиент (ThickClient)** — запуск толстого клиента.

**ТонкийКлиент (ThinClient)** — запуск тонкого клиента.

**Удаление (Delete)** — удаление объекта (основное).

**УправлениеИтогами (TotalsControl)** — управление итогами регистра бухгалтерии и регистра накопления (период итогов, пересчёт).

**Установка (Set)** — установка значения, не сохраняемого в БД.

**Чтение (Read)** — чтение данных объекта из ИБ (основное).

**ЧтениеИсторииДанных (ReadDataHistory)** — чтение записей истории данных.

**ЧтениеИсторииДанныхОтсутствующихДанных (ReadDataHistoryOfMissingData)** — чтение истории данных для отсутствующих данных (удалённые или недоступные по RLS).
