## Сессия приостановлена: universal-xml-exchange2

**Change:** universal-xml-exchange2  
**Срез:** S2 — Полный цикл обмена  
**Причина:** S2.1 BLOCKER — нет `V8Exch82.epf` в репозитории

### 1. Что сделано

S2.1 исследован: в `src/` нет `.epf`, обработки `ргУниверсальныйОбменXML` и макета `ДвижокОбмена`. В `design.md` § Engine Contract зафиксирован BLOCKER и TBD-таблица. Отчёт: `reports/exploration-S2.1-V8Exch82-2026-06-25.md`.

### 2. Что проверить СЕЙЧАС

Ручная проверка не требуется — нужен артефакт epf или конфигурирование S2.2.

### 3. Следующие задачи

| Задача | Действие | Тип | Исполнитель | Статус |

### 5. Blockers

1. **S2.1** — нет `V8Exch82.epf` для static-сверки API; блокирует S2.9, S2.15.

### 6. Issue

Срез S2 нельзя продолжить в BSL, пока не известен публичный API движка. В репозитории есть только эталон cf `УниверсальныйОбменДаннымиXML` (fallback, не epf).

**Options:**

- **A.** Выполнить S2.2 в Конфигураторе: создать обработку `ргУниверсальныйОбменXML`, макет `ДвижокОбмена` (BinaryData) с вашим `V8Exch82.epf`, форму ручного запуска → выгрузить в `src/` → написать «выгрузил S2.2».
- **B.** Положить `V8Exch82.epf` в репозиторий (например `tools/V8Exch82.epf`) → `/opsx:apply universal-xml-exchange2` для повторной S2.1.
- **C.** Явно переключиться на движок cf (Option B) — отдельное решение заказчика + `/opsx:extend`.

### 4. Как вернуться

После A или B: `/opsx:apply universal-xml-exchange2`.

## Ручное конфигурирование: обработка ргУниверсальныйОбменXML

**Конфигурация/расширение:** `рг_УниверсальныйОбменXML`

### Действия в Конфигураторе:

1. Расширение `рг_УниверсальныйОбменXML` → создать **Обработку** `ргУниверсальныйОбменXML` (синоним «Универсальный обмен XML»).
2. Добавить **макет** `ДвижокОбмена`, тип **Двоичные данные**, загрузить файл `V8Exch82.epf`.
3. Создать **форму** (управляемая): реквизит формы — ссылка на `рг_НастройкиОбменаXML` (профиль-приёмник, отбор `Источник = Ложь`); команда «Выполнить обмен» (обработчик — в BSL на S2.17, пока пустой).
4. Выгрузить расширение в `src/ЗУП/cfe/рг_УниверсальныйОбменXML/`.

### После выгрузки

Ожидается: `DataProcessors/ргУниверсальныйОбменXML/`, макет `Templates/ДвижокОбмена/`, `Forms/`.
