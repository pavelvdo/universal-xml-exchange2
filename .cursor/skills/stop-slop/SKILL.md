# Stop Slop (RU)

Убрать предсказуемые AI-паттерны из русскоязычного текста в чате.

## Когда загружать

- Правка шаблонов verify / extend / explore.
- Self-check перед отправкой сообщения пользователю (оркестратор).
- Ревью длинного текста на «воду».

**Не дублирует:** `.cursor/rules/chat-output-budget.mdc` §7 (жаргон движка), `.cursor/docs/chat-lexicon.md` §3 (англицизмы в документах заказчику).

## Отличия от английского stop-slop

| Правило EN stop-slop | RU-адаптация |
|---------------------|--------------|
| No em dashes | **Тире разрешено** — нормативная пунктуация русского |
| Kill all adverbs | Запрещены **пустые усилители** («очень», «крайне», «существенно» без смысла); технические наречия допустимы |
| No Wh- sentence starters | Запрещены **вводные-расшаркивания** («Давайте разберёмся», «Стоит отметить») |
| Active voice only | Запрещена **обезличка** («было принято решение» → «мы решили» / «вы выбрали») |

## Core Rules (кратко)

1. **Без вводных.** Не начинать с «Конечно!», «Отличный вопрос!», «Давайте разберёмся», «Важно понимать».
2. **Без искусственных контрастов.** Не «не просто X, а Y» — сразу Y.
3. **Без пустых усилителей.** «Очень», «крайне», «существенно», «значительно» — только если несут факт.
4. **Без обезлички.** «Было принято решение» → «Вы выбрали» / «План предполагает».
5. **Без мета-комментариев.** Не «в этом разделе мы рассмотрим», «как видно из вышеизложенного».
6. **Без пул-квот.** Если звучит как цитата для LinkedIn — переписать.
7. **Конкретика.** «Последствия значительны» → «Сломается подписание документов без организации».

## Scoring (перед отправкой длинного текста)

| Измерение | Вопрос |
|-----------|--------|
| Прямота | Факты или объявления? |
| Ритм | Разная длина предложений? |
| Доверие | Не сюсюкает читателя? |
| Живость | Звучит по-человечески? |
| Плотность | Есть что вырезать? |

Ниже 35/50 — переписать.
