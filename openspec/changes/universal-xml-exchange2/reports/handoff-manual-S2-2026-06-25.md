# Ручная настройка — срез S2 (блокер перед BSL)

**Change:** universal-xml-exchange2  
**Расширение:** `рг_УниверсальныйОбменXML`  
**Задачи:** S2.2a, S2.3, затем S2.5 (выгрузка), S2.6 (роль)

---

## 1. Форма обработки — S2.2a

**Форма:** `Обработка.рг_УниверсальныйОбменДаннымиXML.Форма.УправляемаяФорма`  
**Конфигурация/расширение:** `рг_УниверсальныйОбменXML`

### Действия в Конфигураторе

1. Открыть расширение → **Обработки** → `рг_УниверсальныйОбменДаннымиXML` → **Формы** → `УправляемаяФорма`.
2. **Реквизиты формы** → Добавить:
   - Имя: `ПрофильПриемника`
   - Тип: `СправочникСсылка.рг_НастройкиОбменаXML`
   - Заголовок: «Профиль приёмника»
   - Связь: при необходимости — параметр выбора / отбор `Источник = Ложь` (только профили-приёмники).
3. **Элементы формы** → добавить поле ввода, привязанное к `ПрофильПриемника` (в удобной группе, например вверху формы).
4. **Команды** → Добавить команду `ВыполнитьОбмен` (синоним «Выполнить обмен»).
5. **Элементы формы** → кнопку с командой `ВыполнитьОбмен`.
6. Сохранить форму.

### После выгрузки

Ожидается в `src/ЗУП/cfe/рг_УниверсальныйОбменXML/DataProcessors/рг_УниверсальныйОбменДаннымиXML/Forms/УправляемаяФорма/Ext/Form.xml`:
- реквизит `ПрофильПриемника`;
- команда и кнопка «Выполнить обмен».

---

## 2. Веб-сервис — S2.3

**Объект:** `rgExchangeService` (SOAP)  
**Конфигурация/расширение:** `рг_УниверсальныйОбменXML`

### Действия в Конфигураторе

1. Расширение → **Web-сервисы** → создать (или открыть) `rgExchangeService`.
2. Добавить **операции**:

| Операция | Параметры (вход) | Возврат |
|----------|------------------|---------|
| `GetData` | `Префикс` (string), `ПрефиксБазыПриемника` (string) | `base64Binary` (или тип Base64 по шаблону платформы) |
| `ConfirmDataReceipt` | `Префикс` (string), `ПрефиксБазыПриемника` (string) | void / boolean по принятому шаблону |

3. Модуль веб-сервиса — пока пустой или заглушка; BSL заполнит задачи S2.11–S2.12.
4. **Публикация** (на ИБ-источнике, администрирование): опубликовать `rgExchangeService`, аутентификация **HTTP Basic** (логин/пароль пользователя 1С с правами обмена).

### После выгрузки

Ожидается каталог:
`src/ЗУП/cfe/рг_УниверсальныйОбменXML/WebServices/rgExchangeService/`  
с `Ext/Module.bsl` и метаданными операций.
