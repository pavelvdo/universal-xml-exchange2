# Шаблон первой строки `/opsx:verify` — формулировки вердикта

Используется **первой строкой** ответа пользователю в чате и **первой строкой** секции «Резюме для разработчика» в файле отчёта.

Главный вопрос: «**могу ли я безопасно запустить `/opsx:apply`?**». В чате имена `GO` / `NO-GO` / `verdict` **не используются**.

**Важно:** первая строка — **вердикт**, не финал сообщения. При GO **обязателен** слот `**Следующий шаг:**` из `chat-summary.md` (GO **не завершает** сообщение).

---

## Вариант 1 — можно запускать apply

```
<change-name> — можно запускать apply.
```

Опционально + одна техническая фраза на языке кода 1С.

---

## Вариант 2 — до старта нужен ваш выбор

```
<change-name> — до старта нужен ваш выбор по логике <тема>.
```

Далее — блок «Что решить» (`templates/chat-summary.md` 3a-decision). **END TURN.**

---

## Вариант 3 — apply пока нельзя (terminal fail repair)

Только когда Repair Loop исчерпан (2 attempts) и repair-блокеры остались:

```
<change-name> — apply пока нельзя: не удалось автоматически дописать постановку за 2 итерации.
```

Далее — 2–3 предложения сути + слот «**Следующий шаг:**» (чат или `/opsx:extend`).

**Не** использовать для промежуточного repair — repair идёт **молча** внутри verify.

---

## Связь с дальнейшим сообщением

После строки вердикта — блоки по `templates/chat-summary.md`:

- **Вариант 1** → резюме + опц. «Подправил в постановке» + **`**Следующий шаг:**`** + ссылка на отчёт.
- **Вариант 2** → триада развилки + **`**Следующий шаг:**`** (ответ в чате) + ссылка на отчёт. **END TURN.**
- **Вариант 3** → 2–3 предложения сути + **`**Следующий шаг:**`** + ссылка на отчёт.

**Удалено:** вариант «нужно дописать постановку» + «допишу через extend» + запуск extend в чате — repair выполняется internal Repair Loop.

---

## Анти-пример

Не использовать:

- «Запускаю `/opsx:extend --from-verify …`»
- GO без `**Следующий шаг:**` + `/opsx:apply` или `/opsx:archive`
- «Что нужно от вас: ничего» на GO без явной команды
- Робот-стек служебных строк: «Что нужно от вас: …» + «Вопрос к вам:» + «Ответьте в чате:»
- «verdict: NO-GO», «Layer 4: CHALLENGE»
