### 1. Что реализовано

В расширении добавлены перечисление статусов и реквизиты профиля (выгрузка S1.1–S1.5). В модуле объекта справочника — проверки при записи по роли (правила, префиксы, адрес/логин) и ограничение регламентного задания профилем-приёмником. На форме элемента — переключение видимости групп по флагу «Источник» и загрузка XML правил в хранилище профиля.

### 2. Что проверить СЕЙЧАС

**Перед тестом в Конфигураторе:** у поля `Источник` на форме элемента привязать событие `ПриИзменении` → `ИсточникПриИзменении`, выгрузить форму.

1. Создать профиль-источник (`Источник = Истина`): загрузить правила XML, заполнить параметры и префиксы → установить статус «Новое» → записать без ошибок.
2. Создать профиль-приёмник с теми же префиксами, адресом и логином → записать без ошибок.
3. Переключить флаг «Источник» — видимость групп и полей приёмника/источника меняется.
4. Запись источника без правил — отказ с сообщением.
