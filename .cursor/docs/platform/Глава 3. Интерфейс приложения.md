# Глава 3. Интерфейс приложения

> Agent ref: Интерфейс: основное/вспомогательное окно, панели, интерфейс Такси.
> Ключевые разделы: 3.2 Окна, 3.3 Закладки, 3.4 Интерфейс Такси
> ---

## 3.1. Общая информация

- Доступ к формам — через команды интерфейса (навигация); структура задаётся подсистемами и объектами метаданных.
- Формы могут отображаться: на начальной странице, в отдельном окне, в закладке, в однооконном режиме (Такси).
- **Режим открытия форм** (окно или закладка) задаётся настройкой клиентского приложения и действует после перезапуска.

## 3.2. Виды окон

- **Основное окно** — навигация и команды. **Вспомогательное окно** — работа с объектами ИБ (документы, элементы справочников, отчёты, обработки); роль вспомогательного может выполнять закладка в рабочей области основного окна.
- Главное меню основных и вспомогательных окон — только общие команды, не настраивается разработчиком.
- **Ограничение:** в режиме отдельных окон каждое окно отображается на панели задач; **веб-клиент этот режим не поддерживает**.

## 3.3. Структура основного окна

- **Панель разделов** — подсистемы верхнего уровня (Рабочий стол + подсистемы). **Панель навигации** — структура по выбранному разделу; гиперссылки открывают формы списков в рабочей области.
- Группы команд: Важное, Обычное, См. также; порядок задаётся при конфигурировании.
- **Панель действий** — команды по текущему разделу (Создать, Отчеты, Сервис и пользовательские группы).
- **Информационная панель** — последние данные и оповещения.
- **Область системных команд** — главное меню, навигация, избранное. **API:** избранное управляется программно через **ИзбранноеРаботыПользователя** из хранилища системных настроек.

## 3.4. Вспомогательное окно и закладки

- Вспомогательное окно используется для работы с одним объектом ИБ (документ, элемент справочника, отчёт). Закрытие окна не закрывает приложение.
- **Ограничение:** для одного объекта ИБ — одно вспомогательное окно (если иное не предусмотрено конфигурацией).
- Можно задать **основную форму** вспомогательного окна.

## 3.5. Интерфейс Такси (однооконный режим)

- Формы открываются в закладках в одном окне; переключение — по закладкам.
- Начальная страница, панель разделов и панель навигации формируются подсистемами.
- **Свойство/параметр:** режим задаётся свойством конфигурации или параметром **MainWindowMode**.
