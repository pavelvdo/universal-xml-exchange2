## ADDED Requirements

### Requirement: Загрузка правил из хранилища профиля

Перед выгрузкой система SHALL извлекать XML правил обмена из реквизита `ПравилаОбменаДанными` профиля-источника, а не из макета конфигурации по имени.

#### Scenario: Выгрузка с правилами из профиля

- **WHEN** выполняется `GetData` для профиля с загруженными правилами
- **THEN** движок обмена использует XML из хранилища профиля

### Requirement: Перевод статуса при выгрузке

При начале `GetData` система SHALL в одной транзакции установить статус профиля «Выполняется». При неустранимой ошибке выгрузки система SHALL установить статус «Ошибка» и записать событие в журнал регистрации.

#### Scenario: Ошибка выгрузки

- **WHEN** выгрузка завершилась ошибкой после перевода в «Выполняется»
- **THEN** статус профиля становится «Ошибка» и в журнал регистрации записывается описание ошибки

## MODIFIED Requirements

### Requirement: Выгрузка через обработку рг_УниверсальныйОбменДаннымиXML

На базе-источнике выгрузка SHALL выполняться встроенной обработкой `рг_УниверсальныйОбменДаннымиXML` расширения: экземпляр создаётся через `Обработки.рг_УниверсальныйОбменДаннымиXML.Создать()`.

#### Scenario: Запуск встроенного движка

- **WHEN** сервер инициирует выгрузку по профилю-источнику
- **THEN** система создаёт экземпляр обработки расширения и выполняет выгрузку через её объектный модуль

### Requirement: Инициализация параметров правил

Система SHALL инициализировать параметры правил обмена значениями из табличной части профиля-источника.

#### Scenario: Подстановка параметров перед выгрузкой

- **WHEN** выгрузка запускается для профиля с заполненной табличной частью параметров
- **THEN** параметры обработки обмена заполняются этими значениями до выполнения выгрузки

### Requirement: Упаковка результата в ZIP

Результат выгрузки SHALL упаковываться в ZIP-архив, содержащий файл XML правил обмена и файл XML данных выгрузки, перед возвратом клиенту через `GetData`.

#### Scenario: Состав архива

- **WHEN** выгрузка успешно сформировала данные
- **THEN** ZIP содержит XML правил из профиля и XML выгруженных данных
