# Глава 29. Система взаимодействия

> **Agent ref:** Система взаимодействия — механизм между ИБ «1С:Предприятие», сервером взаимодействия и внешними системами. Обсуждения (контекстные/неконтекстные), сообщения, видеозвонки. API: глобальный контекст `СистемаВзаимодействия`.
> **Разделы:** 29.1 Общая информация, 29.2 Устройство, 29.3 Программный интерфейс.

---

## 29.1. Общая информация

**Участники:** одна или несколько ИБ (приложение = одна ИБ или область данных), сервер взаимодействия, внешние системы. Приложение идентифицируется однозначно. Пользователи не регистрируются в системе взаимодействия отдельно; авторизация через «1С:Предприятие».

**Режимы взаимодействия:**
- Неинтерактивное — только подсистемы.
- Интерактивное — пользователи: текстовые сообщения, видеозвонки (голос/видео, при полноэкранном окне — демонстрация экрана).
- Смешанное — человек и подсистема (например робот).

**Обсуждения:**
- **Неконтекстное** — не привязано к объекту ИБ; участники задаются вручную; количество не ограничено.
- **Контекстное** — привязано к объекту приложения; участники только с доступом к объекту; у объекта не более одного обсуждения.

К сообщениям можно прикреплять файлы при настроенном внешнем хранилище (протокол Amazon S3). Внешние системы могут работать с обсуждениями через интеграцию.

**Доступ к сообщениям:** Центр оповещений, элемент «Обсуждения» панели открытых, панель сообщений в формах (контекстные), программно — свойство глобального контекста `СистемаВзаимодействия`.

---

## 29.2. Устройство

**Клиент:** часть «1С:Предприятие». **Сервер:** сервис 1С:Диалог или экземпляр 1С:Сервер взаимодействия. Содержимое обсуждений хранится на сервере; данные видеозвонков не хранятся. Обмен с сервером — WebSocket. Видеозвонки — соединение «точка-точка», установка через сервер.

**Прокси:** настройки ОС или inetcfg.xml; NTLM на прокси не поддерживается.

**Видеозвонки:** тонкий клиент — Windows 7+, macOS; веб-клиент — при определённых версиях браузеров (Chrome, Edge, Firefox, Safari) и сервера взаимодействия, только по HTTPS. Трансляция экрана — Chrome, Firefox, по HTTPS.

**Ограничение:** несколько копий одной ИБ не поддерживаются — при копировании возможны общие обсуждения/оповещения и сбои контекстных обсуждений и получателей для объектов/пользователей, созданных после клонирования. Несколько приложений можно объединить в единое пространство (регистрация и объединение с сопоставлением пользователей).

---

## 29.3. Программный интерфейс

Через свойство `СистемаВзаимодействия` глобального контекста доступны:
- регистрация и отмена регистрации приложения;
- объединение и отмена объединения приложений, получение информации об объединённых;
- получение и установка данных регистрации;
- работа с пользователями системы взаимодействия;
- создание и получение обсуждений и сообщений, специализированные инструменты для сообщений.
