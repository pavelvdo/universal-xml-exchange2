# Приложение 12. Описание сущностей, предоставляемых через стандартный интерфейс OData

> **Agent ref:** Сущности стандартного интерфейса OData в 1С:Предприятие.  
> **Ключевые разделы:** OData, константы, справочники, документы, регистры, бизнес-процессы, задачи.

---

## 12.1. Константы

- **Сущность / набор:** `Constant_<Имя>`.
- **Свойства:** Value (тип значения константы), общие реквизиты (кроме независимых разделителей), при отсутствии разделителей — SurrogateKey (Edm.Int16, всегда 0).
- **Поведение:** DELETE не поддерживается. POST: если в БД значение Неопределено — устанавливается; иначе — ошибка. Обращение «через точку» не поддерживается.

## 12.2. Планы обмена

- **Сущность / набор:** `ExchangePlan_<Имя>`.
- **Свойства:** Code (Edm.String/Int32), Description, Ref, DeletionMark, Version, ReceivedNo, SentNo, реквизиты, общие реквизиты, табличные части как отдельные сущности `ExchangePlan_<Имя>_<ИмяТЧ>`.
- **Строка ТЧ:** тип `ExchangePlan_<Имя>_<ИмяТЧ>_RowType`: LineNumber (Edm.Int64), реквизиты, общие реквизиты.

## 12.3. Справочники

- **Сущность / набор:** `Catalog_<Имя>`.
- **Свойства:** Ref, Code (при наличии), Description (при наличии), IsFolder (для иерархии), Owner (для подчинённого), Parent (для иерархии), Version, Predefined (только чтение), PredefinedDataName, реквизиты, общие реквизиты, табличные части как `Catalog_<Имя>_<ИмяТЧ>`.
- **Строка ТЧ:** `Catalog_<Имя>_<ИмяТЧ>_RecordType` / RowType: LineNumber, реквизиты, общие реквизиты.

## 12.4. Документы

- **Сущность / набор:** `Document_<Имя>`.
- **Свойства:** Ref, Number (Edm.String/Int32), Date (при включении), DeletionMark, Version, Posted, реквизиты, общие реквизиты, табличные части как `Document_<Имя>_<ИмяТЧ>`.
- **Методы:** **Post** — проведение (параметр PostingModeOperational, Edm.Boolean, по умолчанию false). **Unpost** — отмена проведения (тот же параметр).
- **Строка ТЧ:** `Document_<Имя>_<ИмяТЧ>_RecordType` / RowType: Ref, LineNumber, реквизиты, общие реквизиты.

## 12.5. Журналы документов

- **Сущность / набор:** `DocumentJournal_<Имя>`.
- **Свойства:** Ref, Number, Date, DeletionMark, Type (Edm.String), Posted, графы, реквизиты, общие реквизиты.
- **Поведение:** только чтение списка; POST и PATCH не поддерживаются.

## 12.6. Планы видов характеристик

- **Сущность / набор:** `ChartOfCharacteristicTypes_<Имя>`.
- **Свойства:** Ref, Code, Description, IsFolder, Owner, Parent, DeletionMark, Version, Predefined, PredefinedDataName, ValueType (EnterpriseV8.TypeDescription), реквизиты, общие реквизиты, табличные части. Строка ТЧ — по той же схеме с суффиксами _RecordType / _RowType.

## 12.7. Планы счетов

- **Сущность / набор:** `ChartOfAccounts_<Имя>`.
- **Свойства:** Ref, Code, Description, OffBalance, Order, Type, Parent, DeletionMark, Version, Predefined, PredefinedDataName, признаки учёта, реквизиты, общие реквизиты, табличные части, ExtDimensionTypes (виды субконто).
- **Строка ТЧ:** стандартная и «Виды субконто» — тип `ChartOfAccounts_<Имя>_ExtDimensions`: LineNumber, ExtDimensionType_Key (Edm.Guid), Predefined, реквизиты, признаки учёта.

## 12.8. Планы видов расчета

- **Сущность / набор:** `ChartOfCalculationTypes_<Имя>`.
- **Свойства:** Ref, Code, Description, ActionPeriodIsBasic, DeletionMark, Version, Predefined, PredefinedDataName, реквизиты, общие реквизиты, табличные части, LeadingCalculationTypes, DisplacingCalculationTypes, BaseCalculationTypes. Составные типы для стандартных ТЧ — по именам суффиксов (LeadingCalculationTypes, DisplacingCalculationTypes, BaseCalculationTypes) с полями LineNumber, CalculationType/CalculationType_Key (Edm.Guid), Predefined.

## 12.9. Регистры сведений

**Независимый:** `InformationRegister_<Имя>`. Свойства: Period (для периодического), SurrogateKey (при отсутствии измерений), измерения (в ключе), ресурсы, реквизиты, общие реквизиты. Изменение ключа — ошибка 404; рекомендуется удалить запись и создать новую. **Функции:** SliceLast(Period, Condition), SliceFirst(Period, Condition).

**Подчинённый регистратору:** `InformationRegister_<Имя>`. Свойства: Recorder, Recorder_Type, RecordSet (Collection). Та же рекомендация по ключу. **Набор записей:** тип строки `InformationRegister_<Имя>_RowType` / `_RecordType`: LineNumber, Active, SurrogateKey (при необходимости), измерения, ресурсы, реквизиты, общие реквизиты. Только чтение (GET).

## 12.10. Регистр накопления

- **Сущность / набор:** `AccumulationRegister_<Имя>`. Recorder, Recorder_Type, RecordSet. Ключ — измерения; при изменении ключа — 404.
- **Функции:** Balance(Period, Condition, Dimensions), Turnovers(Condition, EndPeriod, StartPeriod), BalanceAndTurnovers(StartPeriod, EndPeriod, …). Разворот итогов по периодам не поддерживается.
- **Строка набора:** `AccumulationRegister_<Имя>_RowType` / _RecordType: LineNumber, Active, измерения, ресурсы, реквизиты, общие реквизиты. Только чтение.

## 12.11. Регистр бухгалтерии

- **Сущность / набор:** `AccountingRegister_<Имя>`. Recorder, Recorder_Type, RecordSet.
- **Функции:** Balance(Period, Condition, Dimensions, AccountCondition, ExtraDimensions), Turnovers(…), BalanceAndTurnovers(…), ExtDimensions, RecordsWithExtDimensions(Top, OrderBy, …), DrCrTurnover(…). Разворот по периодам не поддерживается.
- **Строка набора:** LineNumber, Active, Period, RecordType (Debit/Credit), AccountDr_Key, AccountCr_Key или Account_Key, измерения (для небалансовых — суффиксы Dr/Cr), ресурсы (аналогично), реквизиты, общие реквизиты. Только чтение.

## 12.12. Регистр расчёта

- **Набор:** `CalculationRegister_<Имя>`. Recorder, Recorder_Type, RecordSet. При изменении ключа — 404.
- **Функции:** ScheduledData(Condition), ActualActionPeriod(Condition), перерасчёты по имени, Base&lt;ИмяБазовогоРегистра&gt;(Condition, MainRegisterDimensions, BaseRegisterDimensions, ViewPoints).
- **Строка набора:** LineNumber, Active, Period, BegOfBasePeriod, EndOfBasePeriod, BegOfActionPeriod, EndOfActionPeriod, CalculationType_Key, ReversingEntry, измерения (Dr/Cr), ресурсы, реквизиты, общие реквизиты. Только чтение.

## 12.13. Бизнес-процессы

- **Сущность / набор:** `BusinessProcess_<Имя>`. Ref, Code, Description, DeletionMark, Version, Started, Competed, HeadTask (+ _Type), реквизиты, общие реквизиты, табличные части `BusinessProcess_<Имя>_<ИмяТЧ>`.
- **Метод:** Start(RoutePoint — необязательно).
- **Строка ТЧ:** RowType — LineNumber, реквизиты, общие реквизиты.

## 12.14. Задачи

- **Сущность / набор:** `Task_<Имя>`. Ref, Number, Description, DeletionMark, Version, Executed, BusinessProcess (или BusinessProcess_Key), RoutePoint, реквизиты, общие реквизиты, табличные части `Task_<Имя>_<ИмяТЧ>`.
- **Метод:** ExecuteTask.
- **Строка ТЧ:** _RecordType — Ref, LineNumber, реквизиты, общие реквизиты; RowType — LineNumber, реквизиты.
