# Приложение 4. Перечень автоматически сохраняемых настроек

> **Agent ref:** Перечень автоматически сохраняемых настроек платформы 1С:Предприятие. Ключи объектов и настроек в системном хранилище.
> **Ключевые разделы:** автосохранение настроек, хранилище вариантов/настроек отчёта, данные форм, настройки формы, динамический список, системное хранилище.

---

## 4.1. Хранилище: варианты отчёта

- **Ключ объекта:** полное имя отчёта.
- **Ключ настройки:** текстовый идентификатор варианта.
- **Тип сохраняемого значения:** НастройкиКомпоновкиДанных.

---

## 4.2. Хранилище: настройки отчёта

- **Ключ объекта:** полное имя отчёта + «/» + ключ варианта.
- **Ключ настройки:** текстовый идентификатор настройки.
- **Тип сохраняемого значения:** ПользовательскиеНастройкиКомпоновкиДанных.

---

## 4.3. Хранилище: данные форм

- **Ключ объекта:** полное имя формы.
- **Ключ настройки:** текстовый идентификатор сохраняемых значений.
- **Тип сохраняемого значения:** Соответствие. Ключ — текстовый путь к реквизиту данных формы, значение — значение реквизита.

---

## 4.4. Хранилище: настройки динамического списка

- **Ключ объекта:** полное имя формы + имя реквизита типа ДинамическийСписок.
- **Ключ настройки:** текстовый идентификатор сохраняемых значений.
- **Тип сохраняемого значения:** НастройкиТаблицыДинамическогоСписка (в т.ч. ПользовательскиеНастройкиКомпоновкиДанных по контексту). Примечание: в исходной документации тип указан как ПользовательскиеНастройкиКомпоновкиДанных для части настроек.

---

## 4.5. Системное хранилище

### 4.5.1. Настройки формы

**Ключ текущей настройки полей формы**  
Ключ объекта: полное имя формы + «/КлючТекущихНастроекДанных» (/CurrentDataSettingsKey). Если задан КлючНазначенияИспользования: полное имя формы + «/» + значение свойства + «/КлючТекущихНастроекДанных». Ключ настройки — пустая строка. Тип значения — произвольное (ключ текущей настройки).

**Значения полей формы при закрытии**  
Ключ объекта: полное имя формы + «/ТекущиеДанные» (/CurrentData); при КлючНазначенияИспользования — полное имя + «/» + значение свойства + «/ТекущиеДанные». Тип значения — Соответствие: ключ — путь к реквизиту данных, значение — значение реквизита.

**Настройка отображения формы**  
Ключ объекта: полное имя формы + «/НастройкиФормы» (/FormSettings); при непустом КлючНазначенияИспользования — полное имя + «/» + значение + «/НастройкиФормы». Ключ настройки — пустая строка. Тип — НастройкиФормы.

**Размеры формы, элементов, разделители, панель обсуждения, ширины колонок**  
Ключ объекта — полное имя формы + суффикс по режиму совместимости и клиенту:
- 8.3.17 и ранее: не Такси — «/НастройкиОкна» (WindowSettings); Такси — «/Такси/НастройкиОкна».
- 8.3.18 и позже: не Такси — «/НастройкиОкнаТонкогоКлиента» (ThinClientWindowSettings); Такси: тонкий/толстый — «/Такси/НастройкиОкнаТонкогоКлиента», веб — «/Такси/НастройкиОкнаВебКлиента» (WebClientWindowSettings), мобильный клиент — «/Такси/НастройкиОкнаМобильногоКлиента», мобильная платформа — «/Такси/НастройкиОкнаМобильнойПлатформы» (MobilePlatformWindowSettings).  
При заданном КлючСохраненияПоложенияОкна в ключ объекта входит значение этого ключа, например «…/ОсновнойРежим/НастройкиОкна». Ключ настройки — пустая строка. Тип — НастройкиОкна. Сохранение для изменённых форм — каждые 20 минут (тонкий и веб-клиент). Удаление настроек формы выполнять до первого открытия формы в сеансе.

**История поиска в таблице формы**  
Ключ объекта: «Общее/ИсторияПоискаТаблицы/<Имя формы>.<Имя таблицы>» (Common/TableSearchHistory/…). Ключ настройки — пустая строка. Тип — ИсторияПоискаТаблицы.

**История выбора значений в поле ввода**  
Ключ объекта: «ИсторияВыбора» (ChoiceHistory). Ключ настройки — пустая строка. Тип — НастройкиИсторииВыбора.

### 4.5.2. Настройки отчётов

**Ключ текущего варианта отчёта**  
Ключ объекта: полное имя отчёта + «/КлючТекущегоВарианта» (/CurrentVariantKey); при КлючНазначенияИспользования — полное имя отчёта + «/» + значение + «/КлючТекущегоВарианта». Ключ настройки — пустая строка. Тип — произвольное (ключ варианта).

**Ключ текущих пользовательских настроек отчёта**  
Ключ объекта: полное имя отчёта + ключ варианта (строка) + «/КлючТекущихПользовательскихНастроек» (/CurrentUserSettingsKey); при КлючНазначенияИспользования — добавляется «/» + значение свойства. Ключ настройки — пустая строка. Тип — произвольное (ключ настроек).

**Настройки варианта отчёта при закрытии/смене варианта**  
Ключ объекта: полное имя отчёта + ключ варианта (строка) + «/ТекущиеПользовательскиеНастройки» (/CurrentUserSettings); при КлючНазначенияИспользования — полное имя + «/» + ключ варианта + «/» + значение + «/ТекущиеПользовательскиеНастройки». Ключ настройки — пустая строка. Тип — ПользовательскиеНастройкиКомпоновкиДанных.

**Ширина сжатия результата отчёта**  
Ключ объекта: полное имя отчёта + суффикс по клиенту: тонкий/толстый — «/ШиринаСжатияРезультатаТонкогоКлиента», веб — «/ШиринаСжатияРезультатаВебКлиента», мобильный клиент — «/ШиринаСжатияРезультатаМобильногоКлиента», мобильная платформа — «/ШиринаСжатияРезультатаМобильнойПлатформы». Тип — Число.

**Режим отображения результата отчёта**  
Ключ объекта: полное имя отчёта + суффикс. 8.3.17 и ранее: тонкий/толстый/веб — «/РежимОтображенияРезультата». 8.3.18 и позже: тонкий/толстый/веб — «/РежимОтображенияРезультатаТонкогоКлиентаИВебКлиента», мобильный клиент — «/РежимОтображенияРезультатаМобильногоКлиента», мобильная платформа — «/РежимОтображенияРезультатаМобильнойПлатформы». Тип — РежимОтображенияРезультатаОтчета.

### 4.5.3. Настройки динамического списка

**Текущие пользовательские настройки динамического списка**  
Ключ объекта: полное имя формы + имя реквизита ДинамическийСписок + «/ТекущиеПользовательскиеНастройки» (/CurrentUserSettings). При КлючСохраненияПоложенияОкна в ключ входит значение этого ключа. Ключ настройки — пустая строка. Тип — ПользовательскиеНастройкиКомпоновкиДанных.

**Настройки отображения строки поиска конкретного списка**  
Ключ объекта: полное имя формы + имя реквизита ДинамическийСписок + «/НастройкиОтображенияДинамическогоСписка» (/DynamicListViewSettings). Ключ настройки — пустая строка. Тип — НастройкиОтображенияДинамическогоСписка.

**Настройки отображения строки поиска всех динамических списков**  
Ключ объекта: «Общее/НастройкиОтображенияДинамическихСписков» (Common/DynamicListsViewSettings). Ключ настройки — пустая строка. Тип — НастройкиОтображенияДинамическогоСписка.

### 4.5.4. Настройки клиентского приложения

**Настройки клиентского приложения**  
Ключ объекта: «Общее/НастройкиКлиентскогоПриложения» (Common/ClientSettings). Ключ настройки — пустая строка. Тип — НастройкиКлиентскогоПриложения (признаки: отображение команды «Функции для технического специалиста», режим отладки при запуске, низкая скорость соединения для веб-клиента, внешняя диалога получения ссылки; для Такси — состав и расположение панелей).

**Настройки интерфейса клиентского приложения**  
Ключ объекта: «Общее/НастройкиИнтерфейсаКлиентскогоПриложения» (Common/ClientApplicationInterfaceSettings). Ключ настройки — пустая строка. Тип — НастройкиИнтерфейсаКлиентскогоПриложения (расположение панелей).

**Настройки основного окна приложения**  
Ключ объекта по режиму: 8.3.17 и ранее — не Такси «ОсновноеОкно/НастройкиОкна» (MainWindow/WindowSettings), Такси «ОсновноеОкно/Такси/НастройкиОкна». 8.3.18 и позже — не Такси: тонкий/толстый «ОсновноеОкно/НастройкиОкнаТонкогоКлиента», веб «ОсновноеОкно/Такси/НастройкиОкнаВебКлиента»; Такси: тонкий/толстый «ОсновноеОкно/Такси/НастройкиОкнаТонкогоКлиента», веб «ОсновноеОкно/Такси/НастройкиОкнаВебКлиента», мобильный «ОсновноеОкно/Такси/НастройкиОкнаМобильногоКлиента», мобильная платформа «ОсновноеОкно/Такси/НастройкиОкнаМобильнойПлатформы». Ключ настройки — пустая строка. Тип — НастройкиОкна.

**Настройки начальной страницы (рабочего стола)**  
Ключ объекта: 8.3.2 и ранее — «Общее/НастройкиРабочегоСтола» (Common/DesktopSettings); 8.3.3–8.3.7 — «Общее/НастройкиНачальнойСтраницы» (Common/StartPageSettings); 8.3.8 и позже — «Общее/НастройкиНачальнойСтраницы» (Common/HomePageSettings). Ключ настройки — пустая строка. Тип — НастройкиНачальнойСтраницы.

**Положение разделителей начальной страницы**  
Ключ объекта: 8.3.2 и ранее — «РабочийСтол/НастройкиОкна»; 8.3.3–8.3.7 — не Такси «НачальнаяСтраница/НастройкиОкна», Такси «НачальнаяСтраница/Такси/НастройкиОкна»; 8.3.8 и позже — не Такси «НачальнаяСтраница/НастройкиОкна», Такси «НачальнаяСтраница/Такси/НастройкиОкна»; 8.3.18 и позже Такси: тонкий «НачальнаяСтраница/Такси/НастройкиОкнаТонкогоКлиента», мобильный клиент/платформа — аналогично с соответствующими суффиксами. Ключ настройки — пустая строка. Тип — НастройкиОкна.

**Настройки глобального командного интерфейса**  
Ключ объекта (один из): полное имя фрагмента + «/ПанельРазделов/НастройкиКомандногоИнтерфейса» (Common/SectionsPanel/CommandInterfaceSettings); полное имя фрагмента + «/ПанельДействий/НастройкиКомандногоИнтерфейса» (/ActionsPanel/…); полное имя фрагмента + «/ПанельНавигации/НастройкиКомандногоИнтерфейса» (/NavigationPanel/…); «РабочийСтол/ПанельДействий/НастройкиКомандногоИнтерфейса» (8.3.2 и ранее); «ОсновнойРаздел/ПанельДействий/НастройкиКомандногоИнтерфейса» (старше 8.3.2); «РабочийСтол/ПанельНавигации/…» (8.3.2 и ранее); «ОсновнойРаздел/ПанельНавигации/…» (старше 8.3.2). Ключ настройки — пустая строка. Тип — НастройкиКомандногоИнтерфейса.

### 4.5.5. Сервисные настройки

**История глобального поиска** — ключ объекта «Общее/ИсторияГлобальногоПоиска» (Common/GlobalSearchHistory). Тип — ИсторияГлобальногоПоиска.

**История поиска (параметры поиска таблиц)** — ключ объекта «Общее/ПараметрыПоискаТаблиц» (Common/TableSearchParameters). Ключ настройки — пустая строка. Тип — список значений (строки истории, новые в начале).

**Избранное** — ключ объекта «Общее/ИзбранноеРаботыПользователя» (Common/UserWorkFavorites). Ключ настройки — пустая строка. Тип — ИзбранноеРаботыПользователя.

**Настройки справочной системы** — ключ объекта «Общее/НастройкиСправки» (Common/HelpSettings). Тип — НастройкиСправки.

**Цветовые настройки сравнения файлов** — ключ объекта «Общее/НастройкиСравнения» (Common/ComparisonSettings). Ключ настройки — пустая строка. Тип — НастройкиСравнения.

**Настройки печати табличного документа** — ключ объекта «Общее/НастройкиПечатиТабличногоДокумента/» + имя параметров печати. Ключ настройки — пустая строка. Тип — НастройкиПечатиТабличногоДокумента.

**Настройки печати поля HTML-документа** — ключ объекта «Общее/НастройкиПечатиПоляHTMLДокумента» (Common/HTMLDocumentFieldPrintSettings). Ключ настройки — пустая строка. Тип — НастройкиПечатиПоляHTMLДокумента.

**Цвета серий для диаграмм** — ключ объекта «Общее/ЦветаДиаграммы» (Common/ChartColors). Тип сохраняемого объекта — ОформлениеЗначений.

**Параметры окна состояния** — ключ объекта по версии и интерфейсу: не Такси «ОкноСостояния/НастройкиОкна» или «ОкноСостояния/НастройкиОкнаТонкогоКлиента»; Такси — «ОкноСостояния/Такси/НастройкиОкна» или с суффиксами ТонкогоКлиента, МобильногоКлиента, МобильнойПлатформы. Ключ настройки — пустая строка. Тип — НастройкиОкна.

**Параметры окна оповещения** — ключ объекта по версии и интерфейсу: «ОкноОповещения/…» с теми же вариантами суффиксов. Ключ настройки — пустая строка. Тип — НастройкиОкна.

### 4.5.6. Настройки внешних компонент

**Настройки внешней компоненты** — ключ объекта «НастройкиВнешнейКомпоненты/<ИмяНастроекВнешнейКомпоненты>» (AddInSettings/…); имя задаётся компонентой через RegisterProfileAs(). Ключ настройки — пустая строка. Тип — НастройкиВнешнейКомпоненты.

**Параметры подключения управления серверами** — ключ объекта «Общее/ПараметрыПодключенияСерверовАдминистрирования» (Common/AdministrationServersConnectionsParameters). Ключ настройки — пустая строка. Тип — Соответствие: ключ — наименование подключения, значение — Структура с ключами Адрес (Address) — имя или IP сервера администрирования кластера, Порт (Port) — номер порта.
